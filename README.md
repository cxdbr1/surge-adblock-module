# Surge AdBlock Module

Surge 远程模块配置，用于常见广告、跟踪与隐私请求屏蔽。

## 订阅地址

```text
https://raw.githubusercontent.com/cxdbr1/surge-adblock-module/main/adblock.sgmodule
```

## 使用方法

Surge → Modules → Install Module from URL，粘贴上面的远程地址。

## 说明

本模块已为广告/统计相关域名启用 MITM，以支持 HTTPS URL Rewrite；未使用全局 `*`，避免对正常网站造成兼容性问题。
